# JOB_A


## 개요
+ 프로젝트명 : JOB_A
+ 개발기간 : 2020.11.30 ~ 2021.1.18
+ 개발환경 
  - O/S : Windows 10
  - Server : Apache Tomcat 8.5
  - DB : Orarcle 11g
  - Framework/flatform : Spring Tool Suite 4
  - Programming Language : JAVA, JavaScript
  - API : summernote

+ 목적 : 등급제를 도입한 취준생 커뮤니티 

## 내용
+ PPT
+ 구현 기능

  - 사용자
    * 회원가입(이메일 인증)
    * 로그인
    * 게시판(공지사항, 채용공고, 자유, 익명, 멘토&멘티, 후기, summernote, 댓글, 스크랩, 신고)
    * 검색(키워드)
    * 브로드캐스트 채팅

  - 관리자
    * 통계(방문자, 선호직종, 게시글)
    * 회원관리(등급 조정, 활동내역 조회)
    * 게시글·댓글 관리
    * 신고 내역 관리
